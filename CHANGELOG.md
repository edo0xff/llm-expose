# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [0.1.0] - 2026-03-12

### Added
- Initial `llm-expose` CLI release.
- Telegram and Discord client integrations.
- LiteLLM provider integration for 100+ model routing.
- MCP runtime support for tool-aware conversations.
- Pairing workflow for channel access control.
- YAML-based configuration loader and models.
- Baseline test suite for CLI, clients, config, providers, and runtime.
