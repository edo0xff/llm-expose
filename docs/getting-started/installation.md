# Installation

## Requirements

- Python 3.11 or newer
- Access to at least one provider or local OpenAI-compatible endpoint

## Install from source

```bash
git clone https://github.com/edo0xff/llm-expose.git
cd llm-expose
pip install -e .
```

## Install with development tools

```bash
pip install -e .[dev]
```

## Validate CLI

```bash
llm-expose --help
```
